Configuración y conexión a servidor MQTT.

Parámetros
uriDirección del servidor
connCallback de conexión. Se llama cuando el sistema se conecta al servidor.
disconnCallback de recepción. Se llama cuando el sistema se desconecta del servidor.
data_receivedCallback de recepción de datos. Se llama cuando se reciben datos desde tópicos suscritos.
Devuelve
esp_err_t Retorna ESP_OK si todo sale bien.

◆ mqtt_subcribe()

esp_err_t mqtt_subcribe ( char * topic,
int qos )

Suscripcion a un determinado topico.

Parámetros
topicNombre del topico al cual nos suscribimos.
qosCalidad de la suscripcion: 0 - El sistema no verifica que se reciban los mensajes 1 - El sistema verifica que se reciban los mensajes al menos una vez 2 - El sistema verifica que se reciban los mensaje solo una vez y en el orden en el que se enviaron.
Devuelve
esp_err_t Retorna ESP_OK si todo sale bien.
